# 🧠 Multilingual Financial Chatbot (Groq + Firebase + LLaMA 3)

This chatbot allows users to interact in any language and get financial advice powered by the LLaMA 3 model via Groq API. It remembers previous messages using Firebase Firestore and translates conversations between English and the user’s native language.

---

## 🚀 Features

- 🌍 Automatic language detection
- 🌐 Translates user input to English and responses back to user's language
- 🧠 Uses Groq API to access LLaMA 3 for financial Q&A
- ☁️ Stores full chat history per session in Firebase Firestore
- 🔁 Supports multiple sessions with session IDs
